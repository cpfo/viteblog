## 配置

**Run Code 时 OUTPUT输出中文乱码**
最好的解决办法， 新建 系统环境变量 `PYTHONIOENCODING` , 值 为 `UTF-8`

**打开settings.json**
ctrl + shift + P， 输出 `settings.json` 

`Open User Settings (JSON)` 会打开用户设置 settings.json 文件
